# Three Kingdoms: Fortress Arena

삼국지 세계관의 싱글플레이 투기장형 카드 전략 게임입니다.

## 현재 구현 범위 (v0.1)

- 위·촉·오 진영 선택
- 진영별 게임 플레이 정체성 및 군령
- 200종 카드 데이터 카탈로그
- 세 장 중 한 장을 고르는 30회 투기장 드래프트
- AI 30장 자동 드래프트
- 30 내구도의 상·하단 성채
- 1부터 10까지 증가하는 마나
- 후공의 동전
- 턴 드로우, 10장 손패 제한, 7칸 전장, 피로 피해
- 카드 드래그 사용
- 부대 드래그 공격
- 성과 유닛의 피해 수치·흔들림·붉은 파편 연출
- 위/촉/오 군령 프로토타입

## 진영 정체성

- **위**: 속공, 돌진, 직접 피해, 무기, 공격적 템포
- **촉**: 도발, 철갑, 회복, 반격, 장기전
- **오**: 계략, 화공, 광역 피해, 드로우, 발견, 주문 연계
