# Three Kingdoms: Fortress Arena

삼국지 세계관의 싱글 플레이 투기장형 카드 전략 게임입니다.

## v0.2 핵심 구현

- 위·촉·오 진영 선택과 30회 투기장 드래프트
- 정확히 200종의 카드 카탈로그
  - 위 50장 / 촉 50장 / 오 50장 / 중립 50장
- 모든 카드에 서로 다른 자체 생성 SVG 일러스트 적용
- 이미지가 없던 일반 병력·계략·무기·탈것에도 전용 일러스트 적용
- 모든 부대 카드에 키워드, 출전, 유언 또는 지속 효과 부여
- 같은 비용·같은 능력치에서 효과 없는 완전 하위호환 카드 방지 검사
- 진궁·고독한 책사와 허저·호치 등 장수 성향을 반영한 카드 효과
- 상대 AI가 카드를 한 장씩 사용하고 공격하는 순차 애니메이션
- 카드가 손에서 전장으로 이동하는 연출
- 공격 돌진, 공격 궤적, 피격 흔들림, 피해 수치, 파편 효과
- 부대 사망 시 카드가 무너지고 사라지는 전사 연출
- 왼쪽 행동 히스토리 패널
  - 상대 카드 사용
  - 양측 공격
  - 군령 사용
  - 전사 및 턴 시작 기록
- 카드 설명 글자와 카드 확대 비율 개선
- PC 마우스와 모바일 포인터 드래그 지원

## 진영 정체성

- **위**: 돌진, 속공, 성 직접 피해, 처치 보상, 무기
- **촉**: 도발, 철갑, 회복, 흡혈, 재기, 장기전
- **오**: 계략 강화, 화공, 드로우, 은신, 비용 감소

## 기본 규칙

- 성채 내구도 30
- 마나 1부터 시작, 매 턴 최대 마나 1 증가, 최대 10
- 후공은 동전 지급
- 턴 시작 카드 1장 드로우
- 손패 최대 10장, 전장 최대 7칸
- 덱이 비면 누적 탈진 피해

## 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:4173`을 엽니다.

## 검사

```bash
npm test
npm run build
```

## Vercel

저장소를 Vercel에서 Import하면 `vercel.json` 설정에 따라 자동 빌드됩니다.

- Build command: `npm run build`
- Output directory: `dist`

## 저장소 구조

GitHub 연결 도구의 단일 파일 전송 제한을 피하기 위해 실행 소스는 `source-parts/`의 Base64 조각으로 보관됩니다. `npm run prepare-source`가 조각을 결합해 `src/`와 `tests/`의 최신 파일을 재생성합니다. `npm run dev`, `npm test`, `npm run build`는 이 단계를 자동으로 수행합니다.

카드와 UI 이미지는 외부 게임 자산을 복제하지 않고 이 프로젝트 전용 SVG로 제작했습니다.
