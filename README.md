# Three Kingdoms: Fortress Arena

삼국지 세계관의 싱글 플레이 투기장형 카드 전략 게임입니다.

## v0.1 구현

- 위(공격), 촉(방어·회복), 오(계략·화공) 진영 선택
- 정확히 200종 카드와 30회 3택1 드래프트
- AI 30장 덱, 선·후공, 동전, 멀리건
- 성 내구도 30, 마나 1~10, 매 턴 드로우, 손패 10장, 전장 7칸, 피로 피해
- 도발·속공·돌진·은신·철갑·맹독·흡혈·질풍·부활
- 전투의 함성·유언·드로우·발견·침묵·빙결·광역 피해
- 유명 무기와 탈것, 진영별 군령
- 마우스·터치 드래그 공격과 피격 애니메이션

## 실행 및 검사

```bash
npm run dev
npm test
npm run build
```

Vercel에서 저장소를 Import하면 `vercel.json` 설정으로 배포됩니다.
